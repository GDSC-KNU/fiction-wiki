# FictionWiki

위키 시스템을 기반으로 작품들에 대한 정보를 제공하는 웹 플랫폼입니다.

작품 정보, 평점 및 코멘트 등록, 용어집 기반 병렬 번역 기능 등의 각종 서비스와 데이터를 유저에게 통합적으로 제공하는 웹 서비스 제공

<div align="center">

</div>

## 💬 서비스 소개

웹소설, 웹툰 등의 컨텐츠에 대한 위키 서비스를 제공하여 모든 사용자가 문서의 작성에 기여할 수 있습니다.
